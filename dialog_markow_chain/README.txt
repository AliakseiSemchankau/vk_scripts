Скрипт выкачивает какой-либо из диалогов vk и строит по нему марковскую цепь,
генерирующую текст, похожий на осмысленный.

Скрипт msg_downloader.py выкачивает диалог, msg_processor.py обрабатывает файл
и генерирует цепь. Оба скрипта используют файл с параметрами.

В файле parameters.txt содержатся следующие параметры для работы программы:
TOKEN - токен, необходимый для авторизации
chat_id -- номер чата, который нужно проанализировать
msg_store - название файла, в который будет сохранен выкачанный диалог
chain_length - длина марковской цепи, которая будет сгенерирована
3 words - 3 слова, стартовая позиция, с  которой начнет работу марковская цепь. убедитесь,
что такая фраза есть в диалогах
id1 ... - id участников диалога, чьи сообщения учитывать при построении цепь (можно построить
ее как на сообщениях одного человека, так и обоих).
