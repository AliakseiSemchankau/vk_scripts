в parameters.txt:

TOKEN - ключ доступа, который необходим для авторизации
peer_id - id чата, который нужно распарсить
chat_photos.txt - файл, в который будет сохранена промежуточная информация о фотографиях чата
chat_likes_stat.txt - файл, в который будет сохранена обновленная информация о фотографиях чата


chat_picture_loader.py - выгружает всю информацию о картинках в чате в файл chat_photos.txt

chat_picture_like_stat.py - по каждой выгруженной картинке запрашивает количество лайков

photo_like_analyzer.py - подведение итоговых статистик
